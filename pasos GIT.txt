1. crear un proyecto en una carpeta con IntelliJ, esto crea la carpeta del proyecto
2. ir a Git y crear un nuevo repo que sea PÚBLICO y con rama principal "master"
3. abrir nuestra carpeta local y luego una ventana bash-git
4. ejecutar los siguientes comandos:
	- git init
	- git remote add origin <url> -es la misma url que se usa para -git clone
	- git pull origin master --> se decargará el README.md que está en el repo
	- git status --> se verán los archivos del proyecto en rojo
	- git add .
	- git status --> se verán los archivos del proyecto en verde
	- git commit -m:"first commit"
	- git push origin master
